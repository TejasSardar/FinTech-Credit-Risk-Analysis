
# 💳 FinTech Credit Risk Engine

## 📌 Project Overview
This project simulates a bank's risk assessment process. I analyzed ~32k loan records to predict the likelihood of a borrower defaulting.

## 🛠️ Tech Stack
- **SQL:** Data cleaning and Feature Engineering.
- **Python (Pandas, Sklearn):** EDA and Logistic Regression modeling.
- **Excel:** Pivot tables for financial reporting.
- **Power BI:** Interactive risk management dashboard.

## 📈 Key Insights
- Borrowers with a Debt-to-Income ratio > 40% are 3x more likely to default.
- "Medical" loans have the highest default rate compared to "Venture" loans.
- The model achieved an 88% Accuracy in predicting defaults.

## 🚀 How to Run
1. Run the scripts in `/sql_scripts/` to set up the database.
2. Open the Jupyter Notebooks to see the ML pipeline.
3. Open the Power BI file to interact with the data.
